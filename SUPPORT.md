# Support

Thanks for using Examify. Before opening an issue, search the existing issues and check
the setup and environment guidance in [`README.md`](README.md).

Use the issue form that matches your request:

- **Bug report:** reproducible behavior that appears incorrect.
- **Feature request:** a concrete improvement and the problem it would solve.
- **Question:** setup, self-hosting, content-authoring, or usage help.

Examify supports the latest `main` branch only. Community support is best-effort and
has no guaranteed response time.

Please use the correct private or moderated channel for sensitive reports:

- **Security vulnerabilities:** follow [`SECURITY.md`](SECURITY.md) and use GitHub's
  private vulnerability-reporting flow.
- **Conduct concerns:** follow [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md). Never post
  sensitive personal information in a public issue.
